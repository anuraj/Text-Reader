# Text-Reader

It is a file based dotnet windows form app which helps users to read text files.

<img width="1158" height="626" alt="image" src="https://github.com/user-attachments/assets/1beb536d-beee-4322-a3de-1fcc8bfbb9f1" />

## How to run this
1. Clone this repo.
2. Execute the app using `dotnet run Program.cs`.

It is a .NET 10 feature. If .NET 10 not installed, it may not work.
